Tech Stack

Conjunto de tecnologias utilizadas no desenvolvimento dos projetos.
É uma combinação de linguagens de programação, frameworks, estruturas, bibliotecas, padrões, servidores, soluções de UI / UX, software e ferramentas necessárias para o design e desenvolvimentos de aplicações web e mobile e sistemas.

Astah

As ferramentas de modelagem para criação de diagramas UML, diagramas de ER, diagramas de fluxo de dados, fluxogramas, mapas mentais e muito mais.

BootStrap

O Bootstrap é um kit de ferramentas de código aberto para desenvolvimento com HTML, CSS e JS.

Ionic Framework

O Ionic Framework é um kit de ferramentas de interface do usuário de código aberto para a criação de aplicativos móveis e de desktop de alto desempenho, usando tecnologias da Web.

GitHub

GitHub é uma plataforma de hospedagem de código-fonte com controle de versão usando o Git. Ele permite que programadores, utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos privados e/ou Open Source de qualquer lugar do mundo.

Java

O Java é uma tecnologia usada para desenvolver aplicações que tornam a Web mais divertida e útil.

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multi-paradigma.

Jekyll

Jekyll é um gerador de site estático baseado na linguagem Ruby, renderiza Markdown e Líquid. Ele é o mecanismo por trás das páginas do GitHub.

Scratch

Scratch é uma linguagem de programação criada em 2007 pelo Media Lab do MIT. Com ele pode programar as suas próprias estórias, jogos e animações interactivos.
